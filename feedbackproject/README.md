# Feedback Project

This is a Django project for collecting and displaying feedback.

## How to Execute

1. Open your terminal or command prompt.
2. Activate your virtual environment (if applicable):
   ```powershell
   & "c:\Users\Aditya Raj\fsd\.venv\Scripts\Activate.ps1"
   ```
3. Navigate into the project directory:
   ```bash
   cd "c:\Users\Aditya Raj\fsd\feedbackproject"
   ```
4. Run database migrations:
   ```bash
   python manage.py migrate
   ```
5. Start the development server:
   ```bash
   python manage.py runserver
   ```
6. Open your web browser and navigate to `http://127.0.0.1:8000/`.
